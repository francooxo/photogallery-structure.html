# 🖼️ Photo Gallery Structure - Day 4

This project is a structured visual display page built for my Day 4 challenge. It focuses on properly organizing media components layout on the web.

---

## ⭐ Features
* **Media Semantics:** Uses structural `<figure>` tags to handle images cleanly.
* **Captions:** Implements the `<figcaption>` tag to add descriptions directly underneath each photo.
* **Responsive Viewports:** Organized using clean spacing so images adjust layout positions neatly.

## 🛠️ Built With
* HTML5
* CSS3
